# Global Economic Stress Prediction & Monitoring System

## Objective
Build a multi-country macroeconomic stress prediction system using machine learning and MLOps principles.

## Countries
- USA
- Germany
- India
- UK
- Japan

## Features
- GDP Growth (QoQ)
- Inflation (YoY)
- Unemployment Rate
- Interest Rate
- Government Debt (% GDP)

## Project Phases
1. Data Collection & Cleaning
2. Modeling & Evaluation
3. Experiment Tracking (MLflow)
4. API Deployment (FastAPI + Docker)
5. Monitoring & Dashboard

Built as part of advanced data science portfolio development.
